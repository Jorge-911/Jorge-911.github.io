Proyecto preparado para Flask

Inicio

Jorge Tech es un proyecto web orientado a ofrecer servicios tecnológicos de manera clara, confiable y accesible. La interfaz está organizada en bloques reutilizables para facilitar una futura integración con plantillas de Flask.

El encabezado, la navegación, el contenido principal y el pie de página se encuentran identificados mediante comentarios. El contenido repetitivo se genera desde JavaScript utilizando arreglos, objetos y estructuras de repetición.

Quiénes Somos

Jorge Tech es un emprendimiento tecnológico creado para brindar apoyo a personas, estudiantes, hogares y pequeños negocios que necesitan soluciones prácticas en el uso de equipos informáticos y redes. El proyecto ofrece instalación, configuración, mantenimiento y revisión de computadoras, impresoras, cámaras de seguridad, routers y otros dispositivos de uso cotidiano.

La página utiliza una estructura modular que identifica los componentes que posteriormente podrían separarse como archivos de plantilla en Flask, evitando duplicar el encabezado, el menú y el pie de página.
