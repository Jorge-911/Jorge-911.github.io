# Jorge Tech – Semana 5

Proyecto web interactivo desarrollado con:

- HTML5 semántico
- CSS3 externo
- Bootstrap 5
- JavaScript externo
- Manipulación del DOM
- Eventos `submit` y `click`
- `preventDefault()`
- `createElement()` y `appendChild()`
- Validación de campos
- Registro, conteo y eliminación de solicitudes
- Diseño responsivo

## Archivos

- `index.html`
- `style.css`
- `script.js`

## Cómo probar

1. Abra la carpeta en Visual Studio Code.
2. Confirme que los tres archivos estén en la misma carpeta.
3. Abra `index.html` mediante Live Server.
4. Diríjase a **Registro dinámico**.
5. Complete nombre, descripción y categoría.
6. Presione **Registrar solicitud**.
7. Compruebe que el registro aparece sin recargar la página.
8. Use **Eliminar registro** y revise que el contador se actualice.
